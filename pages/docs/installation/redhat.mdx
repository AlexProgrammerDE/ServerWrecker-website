# Installation on Red Hat (.rpm)

Before installing, we recommend you to read the [main installation page](/docs/installation).

## Basic Installation

In this section, you can read how to set up SoulFire on your Red Hat PC.

<Steps>
    ### Download the software
    Download the latest Red Hat installer from [the front page](/).

    ### Run the installer
    To run the installer, open a terminal and navigate to the directory where you downloaded the installer. Then, run the following command:
    ```bash
    sudo dnf install [filename].rpm
    ```

    ### Let the installer do its work
    The installer will now install SoulFire on your system. This should only take a few seconds.

    ### That's it!
    You should now have SoulFire installed on your Red Hat PC. You can now start using it.
</Steps>
