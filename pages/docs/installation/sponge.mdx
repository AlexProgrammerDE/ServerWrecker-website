# Installation on Sponge

Before installing, we recommend you to read the [main installation page](/docs/installation).

## Basic Installation

In this section, you can read how to set up our plugin SkinsRestorer your for Sponge server!

<Callout type="info" emoji="ℹ️">
    If you're using BungeeCord, Velocity (or any fork of those), please follow the [BungeeCord installation guide](/docs/installation/bungeecord)
    or the [Velocity installation guide](/docs/installation/velocity) instead.
</Callout>

<Steps>
    ### Download the plugin
    Download the latest version from [SpigotMC](https://www.spigotmc.org/resources/skinsrestorer.2124/)

    ### Put it in the plugins folder
    Place the `SkinsRestorer.jar` file in the `mods` folder.

    ### Restart your server
    Restart the server. (Only use `/restart` or `/stop` Do NOT use `/reload`)

    ### Configure the plugin
    You may now [configure SkinsRestorer](/setup/configuration#config-guide)

    ### That's it!
    Our plugin should now show at `/sponge plugins list`
</Steps>
