#### Incompatible clients

Launchers that have build in skin mods or other game altering mods that block refresher.

* tlauncher

## general plugin types causing issues

* mostly all other skin plugins like myskin
* Script - they can hook into skins
* small / custom world plugins
* Some TabList plugins

#### Plugins that need config change

* FastLogin -> FastLogin config.yml forwardSkin: false
* MundoSK -> MundoSK config.yml usecustomskins: false
