# Installation on Debian (.deb)

Before installing, we recommend you to read the [main installation page](/docs/installation).

## Basic Installation

In this section, you can read how to set up SoulFire on your Debian PC.

<Steps>
  ### Download the software
  Download the latest Debian installer from [the front page](/).

  ### Run the installer
  To run the installer, open a terminal and navigate to the directory where you downloaded the installer. Then, run the
  following command:
  ```bash
  sudo dpkg -i [filename].deb
  ```

  ### Let the installer do its work
  The installer will now install SoulFire on your system. This should only take a few seconds.

  ### That's it!
  You should now have SoulFire installed on your Debian PC. You can now start the software by searching for it in your
  application menu.
</Steps>
